# **App Name**: Marigo Luxe

## Core Features:

- User Authentication: Secure user authentication with persistent session handling via Firebase Auth.
- Product Listing: Enable sellers to list luxury fashion items with details like price, images, category, and brand.
- Browse and Search: Allow users to browse products, filter them using URL parameters (e.g., category, price), and search for specific items.
- Shopping Cart: Implement a shopping cart to enable buyers to consolidate products, and which can be synchronized to Firestore for logged in users.
- Checkout Process: Guide users through a multi-step checkout wizard (address, payment, review) using react-hook-form and Zod for validation.
- AI Price Suggestion Tool: Use AI tool to suggest a listing price of product to the seller.
- Real-Time Chat: Enable real-time chat between buyers and sellers using Firestore listeners, and real-time notifications

## Style Guidelines:

- Primary color: Purple (#7C3AED) to convey luxury and sophistication.
- Background color: Off-white (#F4F2F9) to create an airy, spacious feel, with hue slightly matched to primary.
- Accent color: Gold (#F59E0B) to highlight key elements like CTAs and special offers.
- Headings: 'Georgia' serif font to reflect elegance and tradition.
- Body: 'Inter' sans-serif font for a clean and modern user experience.
- Logo: 'Poppins' sans-serif font, 700 weight.
- Minimalist design with high whitespace and sharp borders to enhance the airy and luxurious feel.
- Mobile Navigation: Bottom fixed bar (Home, Search, Cart, Favorites, Profile) for easy access on mobile devices.
- Simple, line-based icons to maintain a clean and elegant aesthetic.
- Subtle animations (e.g., fade-in, slide-up) to provide smooth transitions and enhance user engagement.
